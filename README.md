# About The Project

Stolen Roots is a personal digital heritage project created and developed by Zeki M. Cali.
The goal of this project is to show every Somali what was taken from our homeland, where it is being kept today, and how our cultural memory was carried away into foreign museums.

This project is about Greater Somalia, Somali identity, and the objects that belong to our people. Every artifact on this map tells a story of loss, survival, colonial history, and memory. These objects are not just museum pieces. They are part of our roots, our ancestors, our regions, our clans, our cities, and our shared history.

One day, these objects should return to the motherland.
Until that day, this map will help our people see them, remember them, and understand what was taken.

## Focus

This platform currently focuses on:
* Somali objects in foreign museums
* Greater Somalia and Somali cultural regions
* Colonial-era collecting
* Museum records and provenance
* Cultural objects connected to clans, regions, cities, and historical movements
* Public education about stolen and displaced heritage

## How The Map Works

The map shows museums and cultural objects.
Each object can include:
* The object name
* Museum or collection where it is held
* Image
* Source link
* Description
* Provenance or acquisition history
* Coordinates connected to the object’s origin or cultural region

Some objects include multiple images, allowing users to switch between views inside the popup.

## Sources

Most object information comes from museum collection records, including institutions such as:
* British Museum
* Smithsonian National Museum of African Art
* Rautenstrauch-Joest-Museum
* Other public museum collection databases

The project uses these sources to document how museums describe Somali objects and their acquisition histories.

## Why This Matters

Many Somali cultural objects were taken, bought, collected, or removed during colonial rule, military campaigns, foreign expeditions, and unclear circumstances.
For many years, these objects have been hidden in museum databases, far away from the people and communities they came from.

This project brings them back into view.
It is a digital reminder that Somali heritage is not forgotten. It is still connected to the land, the people, and the memory of Greater Somalia. The map exists so Somalis everywhere can see what was taken, learn where it is now, and imagine its return.

This is not only a map.
It is a record of loss.
It is a call for memory.
And one day, it can be part of the road back home.
